# Tabassum Khanum Product Portfolio

Open `index.html` in a browser to view the portfolio.

## Personalize

Search for these placeholders and replace them:

- `Tabassum Khanum`
- `your.email@example.com`
- LinkedIn URL
- Project titles and case study details
- Resume experience and education

The portfolio is static, so it can be hosted on GitHub Pages, Netlify, Vercel, or any simple web host.
